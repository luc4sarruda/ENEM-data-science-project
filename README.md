# ENEM - Data Science & Analysis Insights

## 📌 Project Overview
This repository contains a series of exploratory data analysis (EDA) and data science experiments using the **ENEM 2023** (Brazilian National High School Exam) microdatasets. The goal was to practice data manipulation, visualization, and extract meaningful insights from large-scale educational data.

## 🎓 Acknowledgments & Learning Path
The technical foundation and initial scripts of this project were based on the excellent Data Science course by **Hashtag Treinamentos**:
*   **Source:** [Data Science Playlist - Hashtag Treinamentos](https://youtube.com/playlist?list=PL5TJqBvpXQv5N3iV68bGBkea0HjMk98lR&si=lx7ylF-wJaazBQOL)

While following the course, I implemented my own **personal insights**, modified parameters, and explored different hypotheses beyond the original tutorials to better understand the exam's impact and demographics.

## 🚀 Technologies Used
*   **Python** (Core language)
*   **Pandas** (Data manipulation)
*   **Matplotlib / Seaborn** (Data visualization)
*   **Jupyter Notebooks** (Interactive analysis)

## 📂 Project Structure
*   `analise-dados/projetos/`: Contains the Jupyter Notebooks with the step-by-step analysis.
*   `.gitignore`: Configured to exclude heavy raw data (1.6GB+ CSVs) and local virtual environments.

## 📥 How to Run
1. Download the microdata directly from the [INEP Portal](https://www.gov.br/inep/pt-br/acesso-a-informacao/dados-abertos/microdados/enem).
2. Place the `.csv` file inside a folder named `microdados_enem_2023/`.
3. Install dependencies: `pip install pandas matplotlib seaborn`.
4. Run the notebooks in your preferred environment (VS Code, Jupyter, etc.).
